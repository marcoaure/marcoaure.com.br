Documentation Overview

To edit Nairo Template, basic knowledge about HTML, CSS, JS & Bootstrap is required. You can learn HTML, CSS and JS from W3Schools. You can learn Bootstrap by visiting official site.

All components used in the Template are well commented. If you like to use any component then open the HTML file and refer the code.

Note: Before you make any changes, take a backup of all files and folders.

File Structure

1. Open the Nairo Folder to find all the Template Files
2. You will need to Upload these files to your Web Server using FTP in order to use it on your Website
3. Make sure you upload the required files/folders listed below:
index.html - Index File/Homepage
static/css - Stylesheets Folder
static/img - Images Folder
static/js - Javacripts Folder
static/plugin - Plugin Folder
static/scss - scss Folder(optional)
4. You're now good to go..! Start adding your Content and show off your Beautiful Website in style.

Editing

To edit files you need any code editor. I recommend using Notepad++ and Sublime Text.

Add Custom Styles

To add custom styles, create new style sheet in css folder. Include the style file before the closing tag of <head>. Make sure that you are including this new file after style.css

If you want to add any extra CSS files, copy the file to css folder and link it before style.css.

<link href="static/style/FILE-NAME.css" rel="stylesheet">
Add Custom JavaScript

To add JS, just create a new java script file in js folder. Include the newly created js file before the closing tag of <body>. Make sure that you are including this new file after custom.js. Alternatively you can also include Javascript code directly to the html file by enclosing the code in <script> ...your code... </script> and adding this above the closing tag of <body>

If you want to add any extra JS files, copy the file to js folder and link it after custom.js.

<script src="js/FILE-NAME.js"></script>
Editing HTML

Almost all html codes in this Template are neatly commented. All you need to do is to read the comment and modify the content based on your need.

Fonts Icons

This Template uses Font Awesome Icons and Themify Icons for Icons and Google Web Fonts for Fonts.

Icon Font

Refer Font Awesome Icons and Themify Icons for list of fonts. To use any fonts in the Template, you need to use the following code.

<i class="class-name"></i>

Changing Color Skin

You can change easily color with SCSS variable.

You can set your primary and secondary color in variable.scss file (\static\style\scss)

For Light Theme you need to add theme-light class on body tag 

$px-theme:            #008aff !default;

you can use Koala software for scss compile

Add New Section and Navigation

You wnat to create new section or page you need to follow below points

For Sections
  • 1. Add Link in main navigations 
    <li><a href="#Your_Section_Name">Your Section Name</a></li>
  • 2. Create your section 
    <section id="Your_Section_Name"><div class="container">Your content here</div></section>
For Page
  • This Template has very simple navigation menu. You just need to add unordered list with navigation links for navigation menu. Noting complex. like this 
    <li><a href="/pagename.html">Page Name</a></li>
